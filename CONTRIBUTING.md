# Contributing

Thanks for helping improve `followdraft-skill`.

## Development

Use Node.js 20 or newer, install dependencies, and run the release gate before opening a pull request:

```sh
npm install
npm run release:check
```

Keep changes small and include fixtures or tests when behavior changes.

## Safety

Do not commit private meeting notes, confidential follow-up drafts, customer data, access tokens, or account identifiers. Prefer synthetic inputs that show the workflow without exposing real participants.
